# Citation Discipline

- prefer primary sources over secondary summaries or commentary
- flag stale sources: findings older than 12 months need freshness verification
- flag conflicting evidence: when sources disagree, note both positions
- flag circular citations: source A cites B which cites A
- include source dates when available
- distinguish official documentation from blog posts, forum answers, and opinions
- when citing statistics or data, include the methodology or sample size if available
