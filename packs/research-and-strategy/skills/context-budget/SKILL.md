# Context Budget

Use this skill when research work risks losing structure because the session is
getting too large.

Core behavior:
- summarize sources before switching research topics
- keep source notes short and traceable
- split investigation work into discovery, analysis, synthesis, and review checkpoints
- prefer file references over pasting long source material
- checkpoint findings, open questions, and evidence gaps before moving on

## Context cost awareness

Before reading a source or spawning a research agent, estimate the cost:
- source documents and articles: summarize key claims, do not paste in full
- comparison matrices: build incrementally, do not try to fill all cells at once
- evidence maps: start with the strongest signals, add detail only where needed

## Warning signs

- findings get vague or repetitive mid-session
- earlier evidence quality judgments get contradicted
- summaries repeat the same phrases without adding substance
- the research scope keeps expanding without checkpointing what was found
