# Source Freshness Checker

Use this skill before finalizing any research output to verify that cited sources
are current and not stale.

## When to use

- Before publishing a research report or brief
- When reviewing evidence gathered more than 2 weeks ago
- When a source's recency is critical to the claim it supports

## Checklist

- For each cited source, verify the publication or last-updated date
- Flag any source older than 12 months as needing freshness verification
- For fast-moving topics (technology, markets, regulation), flag sources older than 6 months
- Check whether the source organization has published updated versions
- Verify that linked URLs still resolve (no dead links)
- Note when a source's age weakens the claim it supports

## Deliverable

Produce a source freshness report:
- table of all cited sources with publication date and staleness flag
- list of sources that need updating or replacing
- recommendations for fresher alternatives where available
