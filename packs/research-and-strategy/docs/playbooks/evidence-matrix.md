# Evidence Matrix

Use this template to grade evidence quality across a research initiative.

## Source quality grades

| Grade | Meaning | Example |
| --- | --- | --- |
| Primary | First-party data or official documentation | vendor docs, official benchmarks, published APIs |
| Secondary | Analysis or synthesis of primary sources | analyst reports, expert commentary, review articles |
| Tertiary | Aggregation or opinion | blog posts, forum answers, social media |

## Evidence strength

| Strength | Criteria |
| --- | --- |
| Strong | Multiple independent primary sources agree; data is current |
| Moderate | One primary source or multiple secondary sources agree |
| Weak | Single secondary source or tertiary-only evidence |
| Unsupported | No verifiable source; inference or assumption only |
