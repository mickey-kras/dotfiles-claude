# Context Budget

Use this skill when content work risks losing structure because the session is
getting too large.

Core behavior:
- summarize sources before switching tasks
- keep source notes short and traceable
- split campaign work into research, outline, draft, and review checkpoints
- prefer file references over pasting long source material
- checkpoint decisions, claims, and open questions before moving on

## Context cost awareness

Before reading a source or spawning a research agent, estimate the cost:
- source articles and references: summarize key claims, do not paste in full
- brand guidelines: read once, reference by path afterward
- campaign briefs: read the brief, not the entire background deck
- visual assets: describe what is needed, do not inline image descriptions

Prefer file-path references over pasted content in subagent prompts.
The agent can read the source itself at lower net cost.

## Warning signs

- the editorial angle gets vague mid-session
- earlier decisions about audience or channel get contradicted
- summaries repeat the same phrases without adding substance
- you are tempted to re-read a source you already summarized
