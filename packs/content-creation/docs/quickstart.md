# Content Creation Quickstart

Use this pack for editorial planning, structured research, script development, visual direction, and channel adaptation.

Default flow:
1. Start with the audience, channel, and success metric.
2. Gather source material and separate facts, claims, and open questions.
3. Build the core narrative before adapting it into channel formats.
4. Run an editorial review for claims, citations, brand, and asset provenance.

Profile guidance:
- `focused`: low-risk writing and synthesis.
- `studio`: default for design collaboration and creative iteration.
- `campaign`: trusted-machine mode for broader crawling, search, and asset generation.

Primary playbooks:
- [`editorial-loop.md`](./playbooks/editorial-loop.md)
- [`campaign-build.md`](./playbooks/campaign-build.md)
- [`remotion.md`](./playbooks/remotion.md) — programmatic video with React (requires `ffmpeg` on `$PATH`)
